INTELLIGENT CHAIR


We are Whales. We are a team that consists of young student developers who are currently working on a project for the course II1305. The goal of the project is to develop and assemble a built-in system for chairs which can detect how many people are sitting in a room at any given moment without using cameras. Our customer is professor Mark Smith, who would like to implement this system in Mentorspace which is located in KTH Kista. One of the key challenges of this project is using affordable materials in order to make it scalable as well as overcoming the constraint of long delivery times.
